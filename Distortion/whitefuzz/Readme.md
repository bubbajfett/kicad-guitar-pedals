
# whitefuzz

This pedal is based on the Bearfoot FX artic white fuzz, which is apparently somewhat based on the fuzzface


First build used 2n5457 and while it made a distortion sound, it was
very clippy.  Researching other (dirtboxlayout, effectslayout,
pedalpcb) forums, the recommendation was to use J113 (and a couple
other jfets I did not have).  Replacing this made this sound so much
better.
