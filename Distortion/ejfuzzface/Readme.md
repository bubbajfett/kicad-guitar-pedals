# EJ Fuzz Face

This is a clone of the Eric Johnson fuzz face.   Very nice sound, and gets very
crunchy when first in the chain.

This is the 2nd version of the pedal.   The first version had the fuzz pot without a ground connection 

So this pedal circuit  is verified.  I hate using a 125B box on something
this small but my last attempt at boxing top-jacks on a 1590B did not work very well.
