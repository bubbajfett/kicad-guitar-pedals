# revv-g4


This is a clone of the Revv Amplification G4 overdrive.

This pedal is verified.  It is very loud.   I tested it
10 on the gain, and 2 or 3 on the volume.

The toggle switch does not change the voicing as much as I thought it
would, but overall this gives a nice overdrive/distortion high-gain
pedal to be added to my board.


I have not boxed this one yet.
