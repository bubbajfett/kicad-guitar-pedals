# Fat Drive


This is a clone of the Pigtronice Fad Drive overdrive.  

This also has a very nice crunchy sound.

Changed the board size to 3mm narrower, and that caused the board
to get a new layout that is slightly changed.   Used freerouter with
some manual cleanup.

So this pedal is verified. I have not boxed this one yet, either.
