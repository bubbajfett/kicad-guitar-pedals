# badmonkey


This is a clone of the digitech Bad Monkey overdrive.   This is apparently
a derivative of the tubescreamer pedal from some time ago.   This uses
a gyrator circut for the low's.

Very nice sound.

This is the 2nd version of the pedal.   The first version was missing
the R15/C12.    This was added to the first version of the pedal as a bodge.

So this pedal is verified. I have not boxed this one yet, either.
